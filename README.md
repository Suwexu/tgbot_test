# Telegram Bot для рассылок с кнопками

## Возможности
- 📢 Рассылки в группы и каналы
- 🔘 Интерактивные кнопки с отслеживанием нажатий
- ⏰ Гибкое расписание (ежедневно, интервалы, старт+интервал)
- 📊 Статистика по кнопкам
- 👑 Управление из админской группы

## Установка на Railway
1. Скопируй репозиторий
2. Добавь переменные окружения
3. Разверни

## Переменные окружения
| Переменная | Описание |
|------------|----------|
| BOT_TOKEN | Токен бота от @BotFather |
| ADMIN_ID | Твой Telegram ID |
| TIMEZONE | Часовой пояс (Europe/Moscow) |
| ADMIN_GROUP_ID | ID админской группы (опционально) |

## Команды
- `/admin` - панель управления
- `/start` - добавить группу или начать
- `/id` - узнать ID чата
